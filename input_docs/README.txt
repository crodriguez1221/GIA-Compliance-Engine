Authority Corpus Input Directory

Place raw authority corpus text files here before running the ingestion pipeline.

Examples:

- hipaa_164_308.txt
- hipaa_164_312.txt
- hipaa_164_404.txt
- rcw_19_255.txt
- rcw_42_56.txt
- rcw_70_02.txt
- wac_246_02.txt
- wac_246_10.txt
- watech_policy_1.txt
- watech_policy_2.txt

These files are excluded from version control.

You must obtain official versions of these authorities from their respective
government sources and export them as plain text files before ingestion.

HIPAA: Obtain from the HHS website or ECFR website
RCWs/WACs: Obtain from Washington State Legislature website
WaTech policies: Obtain from WaTech official website
